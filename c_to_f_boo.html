Temperature Converter

Input is in the form of keypad of a calculator and the symbols not only look but the has position similar to that of the device we use everyday, hence this greatly lowers the need to learn this particular input system.



Enter

Enter button is in orange that captures user's attention as it is one of the most frequently used and the important button in the keypad.


It is also slight larger than the rest of the button with consideration of not only its function but the way users interact with the keypad - using the right thumb to click.

Input field is blue to not only stands out in the background but also to differentiate itself from the orange button, thus users can easily spot where they are keying their input.


Also it is in an arrow shape that points to right. The shape can help to guide the user's eyes towards their result which is on the right.

The Enter button is configured that it does all the conversion and update of variables, displaying values.


The "Change" button alternates between the two modes of conversion

The Keypad has a feedback system that when clicked, it informs the user that the button has been clicked.

If user key in too many digits, the interface would remind the person about having too many digits and prompt them to return to the "Neutral" state where they can start their conversion all over.

Depending on the temperature display, the background will change into one of the three background - Hot (when the temperature is above 77F, comfortable between 68F to 77F and cold when it is below 68F). Check out the "Masters" tab to find out the reasonings of the background colours used.
